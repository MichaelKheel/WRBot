World Record Bot

Install plugin

It is only necessary download the plugin and install it on the lan / public server and register in plugins.ini

High-speed performance

The plugin does not a load your lan / public server, loading takes place almost instantaneously.

Show record

It works on all maps where there is a world record.

Play bot

Bot repeats the movement exactly like the demo.

Country flag

Overhead bot flag of the country which currently has the world record.

Update WR

Update occurs automatically after the release sites.

2 Version

We suggest you try the 2 versions of the bot. They differ only by the method of receiving data.
